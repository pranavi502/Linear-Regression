# Linear-Regression
**Linear Regression Project (SLR & MLR):**

This project demonstrates how I implemented Simple Linear Regression (SLR) and Multiple Linear Regression (MLR) using Python.
The goal was to understand the basics of regression, work with real datasets, and evaluate model performance.

**What is Linear Regression?**

SLR: Predicts output using one input variable.

MLR: Predicts output using multiple input variables.
